UK Companies House Query API

This API allows you to query UK Companies House data using SQLite. Available endpoints:

  • /company?q=COMPANY_NAME - Get company details by name.
  • /company-pscs?q=COMPANY_NAME - Get company details and its PSCs.
  • /psc?company_number=12345 - Get PSC details by company number.
  • /psc-company?company_number=12345 - Get PSC and company mapping.

Future version: Convert these responses to BODS statements for open data integration.
